# KursusIn

KursusIn adalah aplikasi berbasis Command Line Interface (CLI) yang digunakan untuk mengelola data peserta kursus digital.

## Fitur

- Menambah peserta
- Mengubah data peserta
- Menghapus peserta
- Menampilkan seluruh peserta
- Pencarian berdasarkan ID (Binary Search)
- Pencarian berdasarkan Nama (Sequential Search)
- Pengurutan berdasarkan ID (Selection Sort)
- Pengurutan berdasarkan Nama (Insertion Sort)
- Statistik jumlah peserta aktif
- Statistik berdasarkan bidang minat

## Bidang Minat

- Backend Engineering
- Frontend Engineering
- Mobile Development
- Data Science & Artificial Intelligence

## Struktur Data

Setiap peserta memiliki informasi:

- ID
- Nama
- Tanggal Daftar
- Bidang Minat
- Status Aktif

## Algoritma yang Digunakan

### Searching
- Sequential Search
- Binary Search

### Sorting
- Selection Sort
- Insertion Sort

## Bahasa Pemrograman

- Go (Golang)

## Cara Menjalankan

Clone repository

```bash
git clone https://github.com/NooelNoa/KursusIn.git
```

Masuk ke folder

```bash
cd KursusIn
```

Jalankan program

```bash
go run main.go
```
